## What is Arqma Wallet

Arqma Wallet is the official wallet for ARQ, a CryptoNote-family privacy cryptocurrency. It's non-custodial, meaning your private keys are generated and stored only on your own device — no account, no company, no server holding your funds on your behalf. The project has no foundation and no venture backing; it's built and maintained by community contributors in the open, on GitHub.

It ships in three forms that all share the same wallet file format and the same 25-word backup seed: a GUI desktop app for Windows, macOS, and Linux; command-line binaries for advanced and scripted use; and native mobile apps for iOS and Android. You can get the current release at [arqmawallet.com/download](https://arqmawallet.com/download).

## Features

| Feature | Description | Learn more |
|---|---|---|
| Non-custodial by design | Keys are generated and stored locally; no third party ever holds them | [How it works](https://arqmawallet.com/how-it-works/non-custodial) |
| On-chain privacy | Ring signatures, stealth addresses, and RingCT hide sender, receiver, and amount | [Privacy details](https://arqmawallet.com/how-it-works/privacy-anonymity) |
| Cross-platform | GUI for desktop, CLI for advanced use, native apps for iOS and Android | [Platform support](https://arqmawallet.com/how-it-works/multi-platform) |
| Local or remote node | Run your own full node for maximum privacy, or connect to a public node for speed | [Node guide](https://arqmawallet.com/how-it-works/nodes) |
| Signed, verifiable releases | GPG-signed releases with published SHA-256 checksums and reproducible builds | [Security review](https://arqmawallet.com/security/is-arqma-safe) |
| Open source | Full source published under the `arqma` GitHub organization | [Open-source details](https://arqmawallet.com/how-it-works/open-source) |
| No telemetry | Official builds send no analytics or device data to any Arqma-operated server | [Privacy notice](https://arqmawallet.com/privacy) |

This repository is an independent documentation resource for Arqma Wallet, a non-custodial, open-source wallet for the Arqma (ARQ) privacy network. It is not a substitute for the official source code, which lives under the [arqma GitHub organization](https://github.com/arqma). For downloads, always use [arqmawallet.com/download](https://arqmawallet.com/download) or an official GitHub release page, and verify checksums before installing.
